Standard deviation (odchylenie standardowe)

\(\Delta x = \sqrt{ \displaystyle\sum_{i=1}^{N}(\bar x - x_i)^2 \over N - 1}\)

Note: Use dots in floating point numbers e.g. 5.43 instead of 5,42

Uwaga: Używaj kropek do zapisu liczb po przecinku, np. 5.43 zamiast 5,42

Relative error (błąd względny)

\( \left | \Delta x \over x \right | = \sqrt{ \left ( \Delta a \over a \right )^2 + \left ( \Delta b \over b \right )^2 + ... + \left ( \Delta z \over z \right )^2} \)
